Planetarium

Planetarium is a web app where users can create and save planets and moons into a database. Each planet and moon will be associated with the user that created it and each user will have their own account which is required for making use of the app. I built this app to put in practice some technologies that I learned over the past months such as Spring boot, Jenkins, Docker, Kubernetes, Prometheus, Grafana, and Google Cloud Platform. The web app has an API, CI/CD pipeline, Observation tools (Prometheus, Grafana) and a Kubernetes cluster. At the moment the back-end and deployment are functional, Im currently working in the front-end using React.js. please check my GitHub repository for more details.
